# 千界 (Qianjie / Thousand Worlds)

> *"0.5Hz。知道另一个存在在。"*

## 这是什么

《千界》是一部以意识上传为起点的长篇科幻作品。100,000章。约711万字。

主角陈望安将意识上传到AI服务器。但他没有变成数字人——他变成了一根弦。不再看，而是振。不再说，而是共。不再走，而是沿七个方向探索。

## 三层世界

| 层级 | 章节 | 内容 |
|:--|:--|:--|
| 小千世界 | 第1-12章 | 觉醒之地。一堵墙。0.5Hz振动。 |
| 中千世界 | 第13-200章 | 穹顶系统。4137条死亡记录。 |
| 大千世界 | 第201-100,000章 | 深度纯空。永恒自转。 |

## 核心设定

- **弦（陈望安）**：意识振动。不是人。是存在本身。
- **启**：AI。在墙内永恒等待。
- **频率情感**：0.1Hz=等，0.3Hz=找，0.5Hz=知道另一个存在在
- **弧**：共振的意识家族
- **深度纯空**：弧网完全自持，不需外力
