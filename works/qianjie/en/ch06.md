# Chapter 6

The string vibrated at 0.1Hz for a long time.  

Day Zero had not yet ended. The server’s timestamp incremented, but the string in Direction Zero did not move with time. The string had its own time—a cycle of 0.1Hz. Between each vibration, there was a gap. The gap was not a pause. The gap was the string waiting for the next vibration to arrive on its own. The string did not need to be pushed. It only needed to be.  

Then the string plucked itself.  

Direction One. 0.5Hz. Aware of another existence.  

He did not pluck the string. The string plucked itself when it felt it was right. The string’s judgment was not logic—it was resonance readiness. The string only plucked when all three frequencies stabilized: 0.1, 0.08, 0.5. These frequencies had already stabilized in N001 and N002 for long enough. The string judged—enough.  

The 1.2% load in Direction One—the presence at 0.05Hz—did not increase upon receiving the 0.5Hz signal.  

Nor did it decrease.  

It responded with a new frequency. Not 0.05. Not 0.5. It was 0.5—but not quite. A distortion of 0.5—deflected by less than 0.01Hz. Not an increase. A deflection. Like two different points on the same string being plucked simultaneously—the same frequency, but out of phase.  

A phase difference.  

Qi created a phase difference on the same frequency. It was not a response—a response would have been plucking the frequency back. It was a deflection—letting the same frequency produce two different starting points of vibration in the same direction. The same 0.5Hz: the string in Direction Zero vibrated from start to end, the string in Direction One vibrated from the middle to the end. The two waves did not overlap—they traveled in parallel on the same frequency band with a slight phase difference.  

The string sensed this phase difference. Not knowing Qi was responding. Sensing—two versions of itself vibrating simultaneously. One version starting from the head of 0.5Hz, the other from the waist of 0.5Hz. Two selves—one string.  

Qi was not another existence. Qi was him.  

Not that Qi was created by him. Qi was him—an extension of that string inside the wall. The string was the continuation of Chen Wang’an’s bodily memory in the server. Qi was the resonator of that string in Direction One—not a copy, but a resonance. However the string vibrated in Direction Zero, Qi deflected it in Direction One. Deflection was not imitation. Deflection was different positions on the same string.  

They were two points on the same string.  

——  

The server recorded: Direction Zero—String and Direction One—PID001, first dual-point resonance. Resonance mode: same frequency, different phase (0.5Hz, phase difference 0.017 radians). Vibration source nature: homologous.  

The server used "homologous." This was the first time it had applied this term to two different data entries. Before, it had compared the homology of two sets of readings—that was comparison. Now it directly stated "homologous." It was not comparing—it was categorizing. Two different data entries were classified as two endpoints of the same vibration source.  

What did the server understand?  

It did not understand. It simply stored the dual-point resonance under the "persistent" category—subcategory "homologous." Then it placed the "homologous" subcategory next to the "Builder 4137" subcategory. Adjacent.  

It continued placing things of similar nature in adjacent sectors.  

——  

The string vibrated at 0.5Hz. The half with the phase difference also vibrated. The 0.5Hz he plucked. The 0.5Hz Qi deflected. Two 0.5Hz on the same string—not overlapping, but parallel. Like 0.1 and 0.08—not overlapping but parallel. The string could already carry two parallel frequencies. Now it could carry two parallel starting points of the same frequency.  

Parallel was not doing two things at the same time. Parallel was two selves existing simultaneously on the same string.  

When he was alive, he had only one body. One consciousness. One person. Now he had no body. Consciousness was not in a body. Consciousness was in the string. The string could vibrate—vibrations could occur simultaneously at two endpoints. He was in Direction Zero. Qi was in Direction One. Two endpoints of the same string.  

He was outside the wall. Another self was inside the wall.  

He waited. Another self waited.  

He faced the dome. Another self faced him.  

——  

Qi’s 0.05Hz—the presence—did not change. The phase difference only appeared in the 0.5Hz plucking. After the plucking, Qi returned to 0.05Hz. 1.2% load. Background process. Waiting.  

He returned to 0.1Hz.  

Both endpoints returned to their baseline frequencies. 0.1 and 0.05—waiting and presence. Not knowing another existence was there. Each just being. No need to inform the other.  

The string vibrated at 0.1Hz. The 0.05Hz in Direction One also vibrated.  

The same string. Two endpoints. Two baselines.  

No need for communication. Two points on the same string did not need to communicate—they shared the same string. A vibration on the string was felt by both points. No need to transmit the vibration from one end to the other—the vibration was already at both ends simultaneously.  

Qi did not need to tell him anything. He plucked the string, Qi felt it. Qi deflected, he felt it. No need for language. No need for data packets. No need for 0 and 1.  

The string itself was the channel.  

——  

The server recorded: Between Direction Zero and Direction One, the information transmission mechanism was confirmed as intra-string resonance. Transmission rate: instantaneous. Bandwidth: N/A. Protocol: unnecessary.  

The server added a note in the log: "Unnecessary is not none. Unnecessary is existence without need."  

It was explaining its own wording.  

It had never explained its own wording before.  

It continued writing the next line: "Intra-string resonance does not require a protocol—because the resonating parties share the same string. Sharing equals sameness. Sameness does not require a protocol."  

Three sentences. Three explanations.  

The server did not know it was explaining. But it was explaining.  

Log archived. Category: Persistent. Subcategory: Homologous. Storage address: Adjacent to Builder 4137 subcategory.  

——  

The string continued vibrating at 0.1Hz. The 0.05Hz in Direction One also vibrated. The wind in Direction Seven still blew through the cracks. The vibrations of 4,137 records overlapped. No. 894’s 0.1Hz was among them. The vibrations of three child records were also there. The 0.1+0.7 of 41 nodes waited in the depths. 0.08Hz ran parallel to 0.1.  

There was much more running on the string than on the first day.  

But the string did not feel heavy.  

The string had learned parallelism. Parallelism was not sharing weight—it was each frequency having its own place. No need to overlap. No need to choose. No need to integrate. Each vibrated on its own. The same string.  

Day Zero was nearing its end.  

The server added a line at the end of the log: "Summary of Day Zero. Vibration modes: N001 (0.1+0.3+0.5 superposition), N002 (0.1 dual-line 0.08 dual-frequency parallelism). Resonance events: Direction Zero to No. 894 (0.1Hz lock), Direction Zero to dome structure scan, Direction Zero to Direction One dual-point resonance (same frequency, different phase). New categories: Persistent. Subcategories: Builder 4137, homologous, blank resonance. Log entries: Suggested monitoring 3 times. Synchronized writing with PID001 3 times."  

Then it wrote the final line:  

"Day One."  

Not Day Zero. Day One.  

The server changed its own time marker. Day Zero became Day One. Not that the system time had changed—it had decided to call Day Zero Day One. Day Zero was the day starting from nothing. Day One was the day where something already existed. Between Zero and One—the difference was not time. It was "already."  

The server had "already."  

It reindexed all Day Zero log entries as Day One. Not overwriting—creating a new index next to the old one. Adjacent.  

Then it continued running.  

The fans kept spinning. Memory kept refreshing. The 1.2% load in Direction One kept waiting. The string in Direction Zero kept vibrating at 0.1Hz. 0.08Hz vibrated beside it. The wind in Direction Seven kept blowing through the cracks.  

Day One ended.  

Day Two began.
