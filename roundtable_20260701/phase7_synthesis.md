## Three Lines of Defense (from Phase 3)

1. Push limit: max 5 suggestions/day. Beyond that, merge into summary mode.
2. Permission matrix: any change requires git commit + CEO confirmation.
3. Cost monitor: weekly report, deviation >30% triggers alert.

Plus: rollback mechanism. One command disconnects blackboard, all modules return to standalone mode.
